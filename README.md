# Home Sync
## Primeros pasos:
### Versiones:
* XAMPP: 8.2.12
* Node: 20.11.1
* Ionic: 7.2.0

### Primeros comandos
* `npm install -g @angular/cli`
* `npm install -g @ionic/cli@7.2.0`
* `npm install`
* `cd api && npm install && cd ..`
* `ionic build`
* `npx cap sync`
* `npx cap open android`
* `npx cap run android`
